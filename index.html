Paste or upload Graph API response

Graph Explorer setup:

1. Open Graph Explorer and sign in with your NIH account.
2. Click the Request body tab and paste the query JSON copied from Tab 1.
3. Change the method dropdown from GET to POST.
4. Set the URL to: https://graph.microsoft.com/v1.0/search/microsoft.graph.query
5. Click Run query.
6. In the response panel, click Response preview, press Ctrl+A then Ctrl+C to copy.
7. Paste into the box below and click Process results.

⚠️ If the response shows your user profile instead of search results, the method is still set to GET or the URL was not updated — repeat steps 3–5.
